崩溃边缘

发表于 2020-04-09 | 分类于 随笔

最近疫情期间出来找工作，虽说国内疫情的情况已经开始好转了很多，但是目前就业方面还是很难找工作，或者说自己好高骛远，眼高手低才没有办法找到工作…

心情低下

今天的心情可以说是极差无比，投简历的公司来电话面试，结果问我的问题基本就两个字 “不会” … 前端就会一个 Vue 框架，后端不会 Node 也不会正统的后端语言，给到的答复是我这种零基础的基本上是很难带的便把电话挂了。

昨天面试的外包公司来电说可以给我去试试，但是经过刚刚电话的面试，我内心对前端的最后一丝自信已经彻彻底底的消失了，便拒绝了试用的工作，或者说之前面试的时候面试题没做几道出来，以及面试我的人对我的态度就像看傻子一样看所以才觉得不想去的吧，更何况公司还很远，便打消了过去的欲望。

现在内心完全是崩溃的…

看房插曲

接完两个公司的电话准备吃饭的时候，说旁边二楼有租房可以去看看房子怎么样，便带着受挫的心情过去了，房东是一个化着很奇怪妆的阿姨，话说的很多还快还听不清楚说什么，带去看房子只有一间房居然租金就已经到了1800，如果我和我妈两个人一起合租都感觉很勉强去付租金，更别说这么小一间房两个人住了。而且没有独立卫生间，不能自己拉网线，这样的房子住着也不方便。

本来心情就不好还有被房东像机关枪一样对着说半天，要是天天有人这样跟我说话，那我不如死了算了！

随便看看

我并不想呆在深圳，没有朋友，而且生活压力那么大，我这种废物可能完全没办法在这种城市活下去吧，我想搬去广州和同学一起合租，但是我搬过去之后要是也找不到工作呢，我走了之后我妈一个人怎么租得起深圳的房子呢？我现在连自己的想法是什么，喜欢做什么，能去做什么我都不知道，完全没有好好的审视过自己，也没有好好的提升过自己，什么都不会却一副高高在上的样子，不会也不愿意跟他人交流，我到底能做好什么？

现在重新开始，还来得及吗？

我这种人，真的可以吗？
